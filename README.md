# _.NoDash()

A fast, searchable reference for vanilla JS alternatives to lodash and underscore.js functions — no dependencies, no build step, single HTML file.

## What it is

NoDash shows you how to replace common lodash utilities with modern JavaScript. Each entry includes the lodash/underscore.js method, a native implementation, and usage notes.

## Usage

Open `index.html` directly in a browser — no server required.

## References

- [You Don't Need Lodash/Underscore](https://github.com/you-dont-need/You-Dont-Need-Lodash-Underscore) — the original inspiration
- [Lodash docs](https://lodash.com/docs) — official API reference
- [MDN JavaScript reference](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) — native method docs

## Stack

- Vue 3 (CDN, no build)
- Single `index.html` — everything in one file
